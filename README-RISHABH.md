## How to Use
1. Add your company logo as `rishabh-logo.png` in the `assets` folder
2. Open `index-rishabh.html` in a web browser
3. Customize content as needed
4. Update contact information if different
5. Replace placeholder blog posts with actual content
6. Update partner logos if needed

## Customization Notes
- To change colors, update the CSS variables in `styles-rishabh.css`
- To modify form submission behavior, update the JavaScript at the bottom of `index-rishabh.html`
- Social media links in footer can be updated in the footer section
- Director images can be replaced with actual photos
